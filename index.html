Pemrograman berbasis web adalah proses membuat sebuah program komputer yang berdasarkan pada halaman situs yang menampilkan berbagai informasi berbentuk teks, grafi, suara dan sumber daya animasi melalui protokol transfer hypertext. Terdapat berbagai macam jenis bahasa pemrograman berbasis web seperti berikut : Bahasa Pemrograman HTML (HyperText Markup Language), Bahasa Pemograman PHP (Hypertext Preprocessor), Bahasa Pemrograman Javascript, Bahasa Pemrograman ASP (Active Server Pages), Bahasa Pemrograman XML (Extensible Markup Language), Bahasa Pemrograman WML (Wireless Markup Language), Bahasa Pemrograman PERL, dan Bahasa Pemrograman CFM.
